## College Attendance Management System
This project is a simple database management system for maintaining student attendance records.
It is built using MySQL and demonstrates the use of relational tables with foreign key constraints.

## Features
Manage student details
Manage faculty details
Assign subjects to faculty
Record and view attendance
Maintain proper database relationships

## Database Tables
students – stores student information
faculty – stores faculty details
subjects – stores subject names and assigned faculty
attendance – stores attendance records of students

## How to Use
Create a new database in MySQL.
Run the SQL scripts to create all the required tables.
Use insert queries to add students, faculty, subjects, and attendance records.
Use select queries to view attendance or student details.
